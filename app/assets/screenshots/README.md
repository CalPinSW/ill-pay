# App Store Screenshots

## Required Sizes

### iOS
| Device | Size | Required |
|--------|------|----------|
| iPhone 6.7" | 1290 x 2796 | ✅ Yes |
| iPhone 6.5" | 1284 x 2778 | ✅ Yes |
| iPhone 5.5" | 1242 x 2208 | Optional |
| iPad Pro 12.9" | 2048 x 2732 | If supporting iPad |

### Android
| Device | Size | Required |
|--------|------|----------|
| Phone | 1080 x 1920 (min) | ✅ Yes |
| 7" Tablet | 1200 x 1920 | Optional |
| 10" Tablet | 1920 x 1200 | Optional |

## Suggested Screenshots

1. **01_home.png** - Receipt list showing multiple receipts
2. **02_receipt_detail.png** - Receipt with items and participants
3. **03_claim_items.png** - User selecting items to claim
4. **04_settlement.png** - Settlement breakdown (who owes what)
5. **05_qr_share.png** - QR code for sharing receipt

## Tips

- Use a clean device with no notifications
- Ensure good sample data is visible
- Consider using device frames for marketing
- Highlight key features with captions

## Naming Convention

```
{platform}/{size}/{number}_{screen_name}.png

Examples:
ios/6.7/01_home.png
ios/6.5/01_home.png
android/phone/01_home.png
```
